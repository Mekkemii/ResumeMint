      Как работает ResumeMint

      ResumeMint - это AI-ассистент для анализа резюме и проверки совместимости с ATS-системами.

      🚀 Основные возможности:

      • Анализ резюме: Структурный анализ, определение грейда (Junior/Middle/Senior/Lead)
      • ATS-совместимость: Проверка оптимизации под системы автоматического отбора
      • Анализ вакансий: Парсинг требований и сравнение с профилем кандидата
      • Рекомендации: Персонализированные советы по улучшению резюме
      • Генерация документов: Редактированное резюме и сопроводительные письма

      🛠 Технологии:

      Использует OpenAI GPT API для анализа и генерации рекомендаций, поддерживает загрузку файлов .docx, .pdf, .txt
